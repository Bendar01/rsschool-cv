### Code Example:  

Bubble sort is a simple comparison‑based algorithm that repeatedly scans a list,
swapping adjacent elements that are out of order.  
After every outer‑loop iteration the greatest unsorted value “bubbles” to its
final position.

```javascript
function bubbleSort(arr) {
  const n = arr.length;
  for (let i = 0; i < n - 1; i++) {
    for (let j = 0; j < n - i - 1; j++) {
      if (arr[j] > arr[j + 1]) {
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
      }
    }
  }
  return arr;
}
```
